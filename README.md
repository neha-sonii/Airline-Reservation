A modern, responsive web application for booking airline flights with an elegant user interface. Built with React and styled with modern CSS, this system allows users to register, login, browse flights, select seats, and manage their bookings.

## üåü Features

### User Authentication

- **User Registration** - Create a new account with email and password
- **User Login** - Secure login for existing users
- **Logout** - Clear user session and return to login
- **Persistent Storage** - User data and bookings saved in localStorage

### Flight Management

- **Browse Flights** - View all available flights with details
- **Flight Information** - Display flight name, date, time, and available seats
- **Seat Selection** - Interactive seat selection with visual feedback
- **Multiple Flights** - Support for multiple flight options

### Booking System

- **Select Seats** - Choose seats from available options
- **View Bookings** - Access "My Bookings" page to see selected seats
- **Booking Details** - Display flight name, seat number, and date for each booking
- **Persistent Bookings** - Bookings saved in localStorage
